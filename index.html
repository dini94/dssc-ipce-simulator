Formula Reference

ParameterFormulaNotes
LHE(λ) LHE(λ) = 1 − 10−f(λ) f(λ) = Gaussian-broadened oscillator strength; Γ = dye surface loading (nmol/cm²)
f(λ) — Gaussian broadening f(λ) = Σᵢ fᵢ × exp(−(λ−λᵢ)²/(2σ²)) Sum over all TD-DFT excited states i; σ = broadening parameter (nm)
Φinj — constant Φinj = constant (user-set) Typically 0.90–0.99 for well-performing dyes (Φinj = ηreg = 1 assumed in Consiglio 2024)
Φinj — from ΔGinj Φinj ∝ f(−ΔGinj) → min(1, −ΔG/(1+|ΔG|)) ΔGinj = E*dye − ECB = −EHOMO − Eex − ECB; must be negative for spontaneous injection
Φinj — Marcus / τ ratio Φinj = 1 / (1 + τinj/τrelax) τinj = 1/kinj from Marcus theory; τrelax ≈ 10 ps (Consiglio 2024)
IPCE(λ) IPCE(λ) = LHE(λ) × Φinj × ηcoll ηcoll = electron collection efficiency; typically 0.80–0.95
Jsc from IPCE Jsc = q ∫ IPCE(λ) × ΦAM1.5G(λ) dλ Integration gives short-circuit current; ΦAM1.5G is the AM1.5G photon flux
APCE(λ) APCE(λ) = IPCE(λ) / LHE(λ) = Φinj × ηcoll Absorbed photon-to-current conversion efficiency; reveals injection×collection efficiency independent of light harvesting

How to Use — Step by Step

Step 1 — Extract excitation data from Gaussian output
Open your .log file, search for Excited State. You will see lines like:
Excited State 1: Singlet-A 3.2156 eV 385.61 nm f=0.7823 <S**2>=0.000
Copy the nm value and f value for each state. Paste into the text area for that dye (one line per state).

Step 2 — Set broadening σ
20–35 nm is typical for organic D-π-A dyes in solution. Increase σ if your experimental spectrum is broader.

Step 3 — Choose Φinj mode
• Constant: simplest — set a single value (0.90–0.99) for all dyes.
• From ΔGinj: provide ΔGinj from your DFT calculation for each dye (negative = spontaneous injection).
• Marcus τ ratio: set τinj and τrelax from your Marcus-theory rates.

Step 4 — Generate and export
Click Generate IPCE Spectra. Switch between IPCE, LHE, Absorption, and Data Table tabs. Export CSV for Excel, PNG for figures, or JSON for further analysis.

Step 5 — Deploy on GitHub Pages
Upload index.html to any GitHub repository → Settings → Pages → Source: main branch root. Your simulator will be live at https://your-username.github.io/repo-name/
