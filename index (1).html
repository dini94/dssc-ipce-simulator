How this tool works & how to use it
This reproduces the author's own theoretical-efficiency pipeline for organic D–π–A dyes on TiO2(101), end-to-end:

1. Absorption spectrum. Each dye supplies either (a) a list of TD‑DFT excited states energy(eV) wavelength(nm) f — one per line — which the tool broadens with a Gaussian band shape in the energy domain (adjustable FWHM) and normalizes to a peak of 1, or (b) a pre-computed continuous spectrum pasted directly as wavelength(nm) value pairs (used for the FPCN/FTCN/FFCN presets, which load the author's exact published normalized-absorbance curves).
2. LHE. LHE(λ) = 1−10−A_norm(λ), with IPCE(λ) = LHE(λ)×IQE (IQE = combined injection×collection efficiency, defaults to 1, i.e. IPCE≡LHE as in the source method).
3. Jsc. The real ASTM G173‑03 AM1.5G spectrum (300–800 nm, embedded) is converted to a photon-flux spectrum in mA·cm−2·nm−1, multiplied by LHE(λ), and integrated (cumulative trapezoid) to give Jsc(λ) and the final Jsc.
4. Voc. From the dye's LUMO and a fixed TiO2 conduction-band edge (ECB), ΔGinj=LUMO−ECB. With a reorganization energy λreorg (input), Voc follows the Marcus-type expression above.
5. J–V curve. J0 is back-solved from (Jsc,Voc) via the single-diode equation, then the curve is built parametrically: at every λ, V(λ)=(kT/q)ln[Jsc(λ)/J0+1] and J(λ)=Jsc−Jsc(λ) — tracing smoothly from (0,Jsc) to (Voc,0), exactly mirroring the spreadsheet method.
6. FF & η. The ideal Green fill-factor FF0 is scaled by a per-dye empirical factor k (series-resistance/real-device correction, ~0.85–0.92); η = JscVocFF/Pin×100 with Pin=100 mW/cm² (1 sun).

Click a preset button to load real data, or add a custom dye and paste your own TD‑DFT excited-state list or spectrum.

Global Simulation Parameters

Integration always uses the full embedded 300–800 nm AM1.5G window; this only zooms the charts.
